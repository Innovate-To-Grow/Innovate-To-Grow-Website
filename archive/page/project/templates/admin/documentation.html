Welcome to I2G

This system is primarily written in Python, JavaScript, HTML, and CSS. It is a Flask application, and the docs are linked below:

Flask docs: https://flask.palletsprojects.com/en/stable/

Running the system for the first time

After cloning the repository to your local machine, it is best to utilize a virtual environment to run the application locally.

python3 -m venv .venv

To start the virtual environment, run:

source .venv/bin/activate

Now you must install all of the dependencies. Run:

python3 -m pip install -r requirements.txt

Now you can run the application:

python3 wsgi.py

Common Error

An error you might encounter is that the gspread config is in the wrong place. Move the file to:

~/.config/gspread/service_account.json

Important Files and Folders

The main folder contains wsgi.py, which is the entry point of the app.

  • config: Contains constants used throughout the application.
  • project: Contains the bulk of the application's code.

Common Tasks

  1. Change home page

    Navigate to project/views/home.py and modify the mainpage function to return the correct template.

  2. Update homepage with final details

    Change times in home-pre-event.html and event.htmlGo to project/templates/home Ensure the event times are updated manually in both HTML files before each event.

  3. Add event to past events

    Update views/home.py by adding a new route for the semester. Copy the current home-post-event.html and create a new template, such as templates/home/2024-fall-event.html. Then update home/past-events.html accordingly by adding a new list element with that event. We also need to make sure that the past event shows the correct events. Go to the new template you made for the past event. In this file, the API url will be present to show the projects for that event. Go to the google sheet for that event and copy and paste the hash after d/ paste that into the hash of the link in the template. Try to match the API call in the template to the sheet. Also if the spreadsheet only has one sheet, add this range A1:Y38 after /values.

  4. Showing current events

    Navigate to the current events template and make sure the API call for the google sheet matches the link of the online spreadsheet. If there is only one sheet in the spreadsheet add this range:A1:Y38
