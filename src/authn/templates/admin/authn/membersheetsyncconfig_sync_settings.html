{# How it works #}

How it works

  • Changes are debounced — rapid edits within 15 seconds merge into one sync.
  • Each sync does a full replace of the sheet (clear + rewrite all rows).
  • Manual sync via admin actions and manage.py sync_members_to_sheet always works regardless of this setting.
