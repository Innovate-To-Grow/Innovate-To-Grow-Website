# Google Sheets Integration

Google Sheets drives several display and import workflows in the ITG platform.

## Guides

- [Display proxy](proxy.md)
- [Database imports](imports.md)
- [Troubleshooting](troubleshooting.md)

## Summary

- `pages` and `events` use sheet-backed display data.
- `projects` can import rows into first-class database records.
- Cache behavior uses a stale-while-revalidate pattern for display endpoints.
